# Java Encapsulation Practice

This project is a simple example of **Encapsulation** in Java.

## Description

The project contains two classes:

### Book_name
This class demonstrates encapsulation by:

- Declaring the `name` field as `private`
- Providing a `getter` method (`getName()`) to read the value
- Providing a `setter` method (`setName()`) to update the value

### Book
This class contains the `main` method and demonstrates how to:

1. Create an object of `Book_name`
2. Set the value of `name` using the setter method
3. Retrieve and print the value using the getter method

## Concepts Used

- Classes and Objects
- Encapsulation
- Private Fields
- Getter Methods
- Setter Methods

## Example Output

```text
Mohammad
```

## Learning Goal

The purpose of this project is to practice Java encapsulation and understand how private fields can be accessed and modified through public methods.
